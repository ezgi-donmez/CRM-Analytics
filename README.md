# CRM Analytics

This repository contains CRM analytics projects developed using Python.  
The main focus is customer segmentation and customer lifetime value prediction using FLO customer data.

## Project Overview

The project includes two main analyses:

1. **RFM Analysis**
   - Customer segmentation based on Recency, Frequency and Monetary metrics.
   - Customers are grouped into behavioral segments such as champions, loyal customers, at risk, hibernating and new customers.
   - Target customer lists are created for marketing campaign scenarios.

2. **CLTV Prediction**
   - Customer Lifetime Value prediction using BG/NBD and Gamma-Gamma models.
   - Expected purchases for 3-month and 6-month periods are estimated.
   - Customers are segmented according to their predicted 6-month CLTV values.
